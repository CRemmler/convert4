GbCC 3.0 Dictionary

NetLogo GbCC 3.0 Dictionary

Overview

GbCC (Group-based Cloud Computing) applications allow users to enter a room where everyone accesses their own instance of a particular NetLogo model. Everyone can share data about their model, in a gallery space.

GbCC Primitives and Commands

gbcc:set

gbcc:set key value

Stores a value, which is accessible by it's key. 

gbcc:set "code" "fd 3"

gbcc:get

gbcc:get key

Returns a value, based on it's key. 

show gbcc:get "code"

gbcc:get-from-user

gbcc:get-from-user user-id key

Returns a value, based on it's user-id and key. 

to gbcc-on-select [user-id]
  show gbcc:get-from-user user-id "code"
end

gbcc:store-globals

gbcc:store-globals

Stores all global values. 

gbcc:store-globals

gbcc:restore-globals

gbcc:restore-globals

Gets stored values and assigns them to their corresponding global variables. 

gbcc:restore-globals

gbcc:restore-globals-from-user

gbcc:restore-globals-from-user user-id

Gets stored values, based on user-id, and assigns them to their corresponding global variables. 

to gbcc-on-select [user-id]
  gbcc:restore-globals-from-user user-id
end

gbcc:broadcast-view

gbcc:broadcast-view name

Sends data to the user's canvas, in the gallery. This data is displayed in everyone's gallery. User can send a screenshot of the view. 

  gbcc:broadcast-view "my-view"

gbcc:broadcast-text

gbcc:broadcast-text name

Sends data to the user's canvas, in the gallery. This data is displayed in everyone's gallery. User can send text. 

  gbcc:broadcast-text "my-text"

gbcc:broadcast-to-plot

gbcc:broadcast-plot name

Sends data to the user's canvas, in the gallery. This data is displayed in everyone's gallery. User can send a screenshot a plot. 

  gbcc:broadcast-plot "my-plot-name"

gbcc:broadcast-avatar

gbcc:broadcast-avatar shape color text

Sends data to the user's canvas, in the gallery. This data is displayed in everyone's gallery. User can send a turtle shape, an integer representing a color and text that can appear below the shape. 

  gbcc:broadcast-avatar "star" 15 "my-name"

gbcc:clear-broadcast

gbcc:clear-broadcast

Remove data the user's canvas, in the gallery. Data is cleared in everyone's gallery. 

  gbcc:clear-broadcast

gbcc:compile-observer-code
gbcc:run-observer-code

gbcc:compile-observer-code key value

gbcc:run-observer-code key

Compile observer code, one time. Run compiled code, repeatedly. 

gbcc:compile-observer-code "code" "show 3"

gbcc:run-observer-code "code"

gbcc:compile-turtle-code
gbcc:run-turtle-code

gbcc:compile-turtle-code who key value

gbcc:run-turtle-code who key

Compile turtle code, one time. Run compiled code, repeatedly. 

gbcc:compile-turtle-code who "code" "fd 5"

gbcc:run-turtle-code who "code"

gbcc:compile-patch-code
gbcc:run-patch-code

gbcc:compile-patch-code pxcor pycor key value

gbcc:run-patch-code pxcor pycor key

Compile patch code, one time. Run compiled code, repeatedly. 

gbcc:compile-patch-code pxcor pycor "code" "set pcolor orange"

gbcc:run-patch-code pxcor pycor "code"

GbCC Reserved Procedures

gbcc-on-enter
gbcc-on-teacher-enter

gbcc-on-enter [ user-id ] gbcc-on-teacher-enter [ user-id ]

User-defined procedure which, if it exists, will be called when any user first enters the room. 

to gbcc-on-enter [user-id]
  show "This is called when a student enters."
  show "This is called when a teacher enters..."
  show"...as long as there is not a defined gbcc-on-teacher-enter procedure."
end

to gbcc-on-teacher-enter [user-id]
  show "This is called when a teacher enters."
end

gbcc-on-go
gbcc-on-teacher-go

gbcc-on-go [ user-id ] gbcc-on-teacher-go [ user-id ]

User-defined procedure which, if it exists, will be called when a user selects a canvas in the gallery, by clicking on it's forever icon. This procedure will be called, continuously, until the canvas is deselected. 

;; runs continuously
to gbcc-on-go [user-id]
  show "This is called when a student's canvas is selected."
  show "This is called when a teacher's canvas is selected..."
  show "...as long as there is not a defined gbcc-on-teacher-deselect procedure."
end
  
;; runs continuously
to gbcc-on-teacher-go [user-id]
  show "This is called when a teacher's canvas is selected."
end
